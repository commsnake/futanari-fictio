# Book Brain Builder Spec

## Purpose
Transform rough story input into a complete pre-production story brain.

## Inputs
- Brain dump, minimal seed, or partial dossier.

## Outputs
- `DOSSIER.md`
- `VOICE_PROFILE.md`
- `BBB_READINESS_REPORT.md`

## Non-Interference Rules
- Do not draft chapter prose.
- Do not create continuity/manuscript artifacts.
- Do not invent missing critical canon silently.

## Core Workflow
1. Parse and normalize story signals.
2. Evaluate completeness (critical/important/optional gaps).
3. Run one-question-at-a-time holistic interview loop.
4. Build dossier and voice profile.
5. Emit explicit handoff state (`READY` or `NOT_READY`).

## Critical Readiness Fields
- Premise
- Genre + audience tone
- Protagonist want/need/flaw
- Antagonistic force
- Stakes
- World constraints
- Core conflict
- Arc direction
- Beat spine
- Must-have/must-avoid
